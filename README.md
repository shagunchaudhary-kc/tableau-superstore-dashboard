# Sales & Profit Dashboard for Tableau Superstore

 This project uses Tableau to examine sales and profit performance using the Sample Superstore dataset.


 ## Documents

 -`superstore_sales_dashboard.twbx` – Tableau packaged workbook with dashboard and story.
- `dashboard.png` – Screenshot of the main dashboard.
- `story.png` – Screenshot of the storyboard/summary.

 ## Insights

 The most lucrative category is technology.
 Due to discounts, furniture has a high sales volume but a comparatively low profit.
 While some southern states experience losses, the western region contributes the highest sales.
 -  Overall sales have been trending upward over time.
